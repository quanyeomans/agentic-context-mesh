:::note Enable the Fusion readiness panel

The <Constant name="fusion" /> readiness panel shows each project's eligibility status and blockers in the <Constant name="dbt_platform" />. It's rolling out in phases &mdash; if it's not enabled for your account yet, an [account admin](/docs/cloud/manage-access/enterprise-permissions#account-admin) can turn it on in **Account settings** → **Account**. Refer to [Enable Fusion readiness features](/guides/prepare-fusion-upgrade?step=2) for setup steps.

If you have access to <Constant name="copilot"/>, use the [<Constant name="dev_agent" />'s <Constant name="fusion" /> migration workflow](/docs/dbt-ai/developer-agent#fusion-migration-workflow) to help you fix compatibility errors directly from the <Constant name="studio_ide"/> using <Constant name="copilot"/> &mdash; no manual log investigation needed!

:::
